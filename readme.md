## Helper for scss project 

* Font size
* Simple grid
* Simple visual design
* Color helper


## Compile and watch
 I'm using node-sass to compile and watch scss files.
#### compile: 

``` node-sass src/index.scss dist/style.css ```


#### watch:

``` node-sass src/index.scss dist/style.css -w ```
